## 🚀 Getting Started

### Prerequisites
No build tools or dependencies are required — a modern web browser is all you need.

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Samuel-Mbah/resume.git
   cd resume
   ```

2. **Open in browser**
   ```bash
   # macOS
   open index.html

   # Linux
   xdg-open index.html

   # Windows
   start index.html
   ```

### Deploy to GitHub Pages

1. Push this repository to GitHub (already done).
2. Go to **Settings → Pages**.
3. Under *Source*, select the branch (e.g. `main`) and `/ (root)`.
4. Click **Save** — GitHub will publish the site at `https://<username>.github.io/resume/`.
